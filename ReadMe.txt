## 内部対策を考慮したWordpressテーマ

前回製作したWordpressテーマは、概ねUdemyの教材に沿って制作したものでした。

しかしその後SEOの内部対策などについて調べる中で「SEO対策を意識したテーマを作ってみたい」と思いました。

そこで今回は、「表示の速さ」「内部対策の強化」「読みやすいレイアウト」といった部分を意識して制作を進めました。

## 具体的に取り組んだ点

- 構造化マークアップ
- タグの最適化
- リクエスト回数を最小限にする
- なるべくプラグインに頼らない（パンくずリストなどの自作）
- 読みやすいレイアウト
- レスポンシブ対応
- GoogleのPageSpeed Insightでの速度チェック、改善
- W3CでHTML構造のチェック
- Scssの導入（前回はCSSのみで作成していたため）
